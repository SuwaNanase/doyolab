だが、

Raspberry Piでプログラムを自動起動する5種類の方法を比較・解説を参照に行った

目のプログラムと簡単な時刻のプログラムは実行できた。



その後も何度か別のサイトを参照にしながら行ったりもしたが、できない。



原因がわからないまま終わった。







✳︎目のプログラム

  1. $ mkdir -p ~/.config/lxsession/LXDE-pi
  2. $ cp /etc/xdg/lxsession/LXDE-pi/autostart ~/.config/lxsession/LXDE-pi/
  4. $ sudo apt-get update
  5. $ sudo apt-get install x11-apps
  7. $ echo '/usr/bin/xeyes' >> ~/.config/lxsession/LXDE-pi/autostart




✳︎時刻のプログラム

  1. $ sudo nano /etc/rc.local
  2. $ sudo systemctl daemon-reload
  3. $ sudo systemctl start test.service
  4. $ sudo systemctl status test.service
  5. $ sudo systemctl stop test servise
